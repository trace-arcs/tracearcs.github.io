Introduction to Mobile AR in Unity

This Programming Historian tutorial serves as an introduction to creating mobile augmented reality applications using the Unity game engine and Vuforia SDK. In this tutorial, you will learn how to navigate the Unity interface, convert images to augmentable data sets, add image overlays, and build standalone apps to Android and iOS devices.

by Jacob Greene
